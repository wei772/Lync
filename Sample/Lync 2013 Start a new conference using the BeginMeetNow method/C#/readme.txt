MeetNow Sample

This sample uses the Lync Model API to add or remove custom groups.

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\MeetNow
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\MeetNow

Features
- Start a MeetNow conference using Lync Automation class

Warnings:
- The sample runs in Visual Studio 2010.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 4.0 or above.
- Visual Studio 2010 or above.
- Microsoft Lync SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync must be installed and running.

Running the sample
------------------
1. Open MeetNow.csproj
2. Hit F5 in Visual Studio
